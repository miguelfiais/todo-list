## Visão geral

![todoGit](https://user-images.githubusercontent.com/108070001/219745443-8c8ca518-ee66-4ffd-b557-6961eb343159.gif)

Refiz minha lista de tarefas com o intuito de aplicar meus conhecimentos em front-end e tornar este projeto mais útil no dia a dia. Para isso, salvei as tarefas no local storage para que não desapareçam quando o usuário atualizar a página e utilizei o hook useContext para manipular os dados, tornando a aplicação mais prática. Além disso, também utilizei o react-router-dom para permitir a mudança entre as opções de exibição 'all', 'active' e 'completed'.

## Como usar


Para clonar e executar este aplicativo, você precisará [Git](https://git-scm.com) e [Node.js](https://nodejs.org/en/download/) (que vem com [npm](http://npmjs.com)) instalado no seu computador. Na sua linha de comando:

```bash
# Clonar este repositório
$ git clone https://github.com/miguelfiais/todo-list

# Instalar dependências
$ npm install

# Execute o aplicativo
$ npm start
```
